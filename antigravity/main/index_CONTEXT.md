# Home Page (`index.html`) 開発コンテキスト

本ドキュメントは、トップページ (`index.html`) の開発ガイドラインです。

## 1. 概要

- **役割**: アプリの玄関口（ランディングページ）。
- **目的**: ユーザーに「祭りの高揚感」を与えつつ、必要な情報へ直感的にアクセスさせる。
- **デザイン**: Vibrant（鮮やか）かつ Glassmorphism（すりガラス）を取り入れたモダンな UI。

## 2. 機能・コンテンツ詳細

### 2.1 ヒーローセクション (Hero Section)

- **ビジュアル**: 没入感のあるグラデーション背景、浮遊するオーブ装飾。
- **コンテンツ**:
  - イベントバッジ: 日付などを表示。
  - **大見出し**: `Nanryosai 2026`（グラデーション文字）。
  - スローガン: 今年のテーマ。

### 2.2 カウントダウン (Countdown)

- **機能**: 開催日（2026 年 9 月 11 日 9:00）までの残り時間をリアルタイム表示。
- **デザイン**: Glassmorphism スタイルのカードに Days / Hours / Minutes / Seconds をグリッド表示。

### 2.3 リアルタイム情報 (Real-time Info)

- **現状**: **Dummy UI** として実装済（2025/01/09 時点）。
- **内容**: 体育館、音楽室、視聴覚室の 3 会場の状況を表示。
- **レイアウト**: 横一列のカード配置。
- **将来の実装予定**:
  - **Cloud Functions + Storage**: 管理画面から JSON を更新し、読み込む仕組みを採用予定。
  - **Admin**: パスワード認証付きの専用更新ページ (`admin/stage-editor.html`) を作成予定。

### 2.4 ダッシュボード (Shortcuts) - _Coming Soon_

- **企画一覧**: 参加団体の出し物一覧へ遷移。
- **ステージ**: タイムテーブルへ遷移。
- **モバイルオーダー**: 模擬店の注文ページへ遷移。

### 2.5 モバイルオーダー推奨 (Mobile Order Promo)

- **役割**: モバイルオーダー機能（`pos/mobile-order.html`）への誘導。
- **デザイン**: 鮮やかなグラデーションと Glassmorphism を用いた強調カード。
- **CTA**: "Start Ordering" ボタンでオーダー画面へ遷移。

### 2.6 企画・発表ハイライト (Highlights)

- **役割**: ユーザーに「面白そう」と思わせる興味喚起エリア。
- **デザイン**: 横スクロール (Infinite Loop)。
- **コンテンツ**: 各クラスや部活の企画カード（カテゴリ、名称、場所）。
- **実装**: `main/data/data.js` から `projectData` を読み込み、JS で動的に生成。
- **インタラクション**: マウスドラッグおよびタッチ操作による横スクロールが可能。

### 2.7 データ管理 (Data Management)

- **ファイル**: `main/data/data.js`
- **内容**: 企画データ (`projectData`) やステージデータ (`stageData`) を一元管理。
- **目的**: HTML とデータを分離し、更新性を向上。

## 3. 実装要件

- **App Shell 統合**: `app-shell.js` によりヘッダー・フッターを動的に注入。
- **レスポンシブ**: モバイルファーストだが、PC 閲覧時も間延びしないよう `max-width` などを適切に設定。
- **CSS**: `style.css` にて一元管理。変数を活用し、ダークモードにも完全対応する。
