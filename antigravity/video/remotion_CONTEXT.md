# Remotion Context

## 概要

インフォグラフィックアニメーション動画を作成するための Remotion プロジェクト。
Webサイトのコードベース (`main`, `pos`) とは独立しており、`video` ディレクトリ配下で管理される。

## 環境

- **ディレクトリ**: `video/`
- **フレームワーク**: Remotion (Reactベース)
- **テンプレート**: `helloworld`

## 開発コマンド

`video` ディレクトリ内で実行すること。

| コマンド        | 説明                                                         |
| --------------- | ------------------------------------------------------------ |
| `npm run dev`   | Remotion Studio (プレビュー) を起動 (デフォルトポート: 3000) |
| `npm run build` | 動画のレンダリング (MP4出力)                                 |

## ディレクトリ構成

```
video/
├── src/           # 動画のソースコード (Reactコンポーネント)
├── public/        # 静的アセット (画像、音声等)
├── package.json   # 依存関係
└── remotion.config.ts # Remotion設定
```

## 注意点

- Webサイト側のCSSやJSとは分離されているため、サイトと同じデザインを適用する場合はデザインシステム（CSS変数や画像アセット）の共有方法を検討する必要がある。現状は独立している。
