# 南陵祭 2026 団体提出データ要件

南陵祭の「Web サイト（広報）」および「モバイルオーダーシステム（注文）」の運用に必要な情報項目リストです。括弧内はプログラムやデータベースで使用されるシステムキー名です。

---

## 1. 共通基本情報（全団体必須）

| 項目名               | システム項目名 ([data.js]/Firestore) | 説明・例                                   | 備考                                       |
| :------------------- | :----------------------------------- | :----------------------------------------- | :----------------------------------------- |
| **団体名・クラス名** | `groupName` / `name`                 | 組、部活名（例：`3年1組`、`美術部`）       | **2026 版から項目が分離**                  |
| **企画名・店名**     | `name` / `teamName`                  | お店や出し物の名前（例：`激ウマ焼きそば`） |                                            |
| **活動場所**         | `place`                              | 教室、テントの場所（例：`北棟 3F 301`）    |                                            |
| **階数**             | `floor`                              | 開催場所の階数（例：`1`、`2`、`3`）        |                                            |
| **マップ座標**       | `mapX`, `mapY`                       | デジタルマップ上での表示位置（数値）       | **昨年使用。今年も地図機能を使う場合必須** |

## 2. Web 広報用データ（全団体必須）

| 項目名               | システム項目名 ([data.js]/Firestore) | 説明・例                                   | 備考                           |
| :------------------- | :----------------------------------- | :----------------------------------------- | :----------------------------- |
| **キャッチコピー**   | `catchphrase`                        | 短い宣伝文句（例：`伝統のソース味！`）     | 20 文字程度推奨                |
| **紹介文**           | `description`                        | 詳細ページの PR 文。(PR ポイントなど)      | 100〜200 文字                  |
| **タグ**             | `tags`                               | 検索用のカテゴリタグ（例：`食品`、`展示`） | 複数選択可                     |
| **メイン画像**       | `image`                              | 看板となる画像ファイルパス                 | 横長推奨                       |
| **コンテンツタイプ** | `contentType`                        | 企画の種類（`menu` または `gallery`）      | `menu`：販売 / `gallery`：展示 |

## 3. モバイルオーダー用商品データ（食品・物販団体のみ）

| 項目名            | システム項目名 (Firestore) | 説明・例                                     | 備考                                  |
| :---------------- | :------------------------- | :------------------------------------------- | :------------------------------------ |
| **メニュー名**    | `name`                     | 商品名（例：`ソース焼きそば`）               |                                       |
| **価格**          | `price`                    | 税込価格（数値。例：`300`）                  |                                       |
| **商品説明**      | `description`              | 商品ごとの詳細説明（例：`秘伝のソース使用`） | **Firestore も `description` で統一** |
| **商品画像**      | `imageUrl`                 | 商品の画像 URL                               | **必須** (Firebase Storage の URL)    |
| **トッピング/OP** | `allowedToppings`          | 選択肢（例：`大盛り`, `マヨ抜き`）           | 配列形式                              |
| **おすすめ**      | `isRecommended`            | `true` の場合に Web で強調表示               |                                       |
| **販売中フラグ**  | `isAvailable`              | `true` で注文可能、`false` で売り切れ表示    |                                       |

## 4. その他（展示・ステージ）

### 展示・アトラクション

| 項目名       | システム項目名 ([data.js]/Firestore) | 説明                               |
| :----------- | :----------------------------------- | :--------------------------------- |
| **追加画像** | `gallery`                            | 紹介文の下の追加写真（パスの配列） |

### ステージ発表

| 項目名         | システム項目名 (`stageData`) | 説明                                                   |
| :------------- | :--------------------------- | :----------------------------------------------------- |
| **出演日時**   | `time`                       | 出演枠（例：`Day1 09:30 - 10:30`）                     |
| **出演団体名** | `groupName`                  | ステージ上の団体名                                     |
| **演目名**     | `name`                       | ステージで発表する内容名                               |
| **紹介文**     | `description`                | ステージ発表の紹介（昨年の `groupDescription` に相当） |

---

## 昨年（2025 年）データとの主な変更点

- **団体名と企画名の分離**: 昨年は `name` に「1 年 1 組 タケムランド」のように両方入っていましたが、今年は `groupName` と `name` を分ける構成になっています。
- **マップ座標の不在**: 昨年は `mapX`, `mapY` がありましたが、今の [data.js] にはありません。今年も地図機能を使うなら追加が必要です。
- **商品詳細 (`desc`)**: 昨年は商品ごとに `desc` がありましたが、今の [data.js] のサンプルにはありません。
